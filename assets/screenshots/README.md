# Screenshots

This directory contains screenshots showcasing Orion AI's features.

## Required Screenshots

To complete the documentation, please add the following screenshots:

1. **main-interface.png** - Main chat interface showing:
   - Sidebar with conversation list
   - Chat area with messages
   - Model selector in header
   - Message input area
   - Dark theme with purple accents

2. **model-selector.png** - Model selection dropdown showing:
   - Search functionality
   - Featured models section
   - Model cards with provider, context window, and pricing
   - Refresh button

3. **mcp-servers.png** - MCP server management view showing:
   - List of configured servers
   - Status indicators (running/stopped)
   - Start/stop controls
   - Add server button

4. **smithery-browser.png** - Smithery marketplace dialog showing:
   - Search and filter functionality
   - Featured servers section
   - Server cards with descriptions
   - Install buttons

5. **settings.png** - Settings dialog showing:
   - API key configuration
   - Model selection
   - MCP servers tab
   - Keyboard shortcuts

## How to Take Screenshots

1. Run the app: `npm run tauri dev`
2. Navigate to each view mentioned above
3. Take high-quality screenshots (1920x1080 or higher recommended)
4. Save with the exact filenames listed above
5. Ensure dark theme is active for consistency
6. Crop to remove unnecessary window chrome if desired

## Screenshot Guidelines

- **Resolution**: At least 1920x1080 for clarity
- **Format**: PNG for best quality
- **Theme**: Dark mode with purple accents
- **Content**: Show realistic usage examples, not empty states
- **Privacy**: Don't include sensitive information (API keys, personal data)
